<h3>Self-test: Send WADO Retrieve (RAD-55) Request to IDS sim.</h3>

<p/>When the test section is run:
<ol>
<li/>The Image Document Consumer simulator (IDC sim) sends WADO Retrieve 
(RAD-55) Requests to the Imaging Document Source System Under Test (IDS SUT).
<li/>The IDS SUT is expected to process the requests and generate responses
containing the requested images.
<li/>The IDC sim will process the RAD-55 responses and store the received
images.
</ol>
