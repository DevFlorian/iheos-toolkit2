Merge PID/XAD-PID to Different PID/XAD-PID: Two Documents and Folder
<br/>
All objects are moved to the new local PID/XAD-PID.
<br/>
1) Generate and register an initial XAD-PID used for the submission. <br/>
2) Submit two DocumentEntry object for the initial Local PID/XAD-PID. <br/>
3) Generate and register the new XAD-PID that will be used for the merge. <br/>
4) Execute the Notify XAD-PID Link Change transaction to merge local PID/XAD-PID under a new local PID/XAD-PID combination. <br/>
5) Verify the documents for local PID were merge by the Document Registry under the new PID/XAD-PID. <br/>
