Link Change: Single Document

Local PID Moved From XAD-PID1 to XAD-PID2<br/>
</br>
1) Generate and register an initial XAD-PID used for the submission. <br/>
2) Submit a single DocumentEntry for the initial XAD-PID. <br/>
3) Generate and register the new XAD-PID that will be used for local PID move. <br/>
4) Execute the Notify XAD-PID Link Change transaction to move the local PID. <br/>
5) Verify the documents for local PID were moved by the Document Registry. <br/>
