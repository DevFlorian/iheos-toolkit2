ITI-62: Remove DE0:APND:DE1 DE0:XFRM_RPLC:DE2 Objects

This test evaluates the Document Registry's enforcement of the Document Sharing Metadata Model. 

Three submissions are made in this test: 1) an original DocumentEntry, 2) an appended DocumentEntry object to the original, and 3) a replacement for the original DocumentEntry.  

Test include removing the association and each submission set individually. These test should fail. 

The final test should successfully remove all submitted objects.
