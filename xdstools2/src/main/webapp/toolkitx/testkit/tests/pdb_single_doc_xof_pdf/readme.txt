Send Provide Document Bundle in XDS on FHIR context (PDF)

Sends a Provide Document Bundle transaction (ITI-65) to an (MHD) Document Recipient which is
bound to an (XDS) Document Source as specified in the MHD XDS on FHIR option. The Document
Recipient/Document Source is the System Under Test (SUT).  The (XDS) Document Source must be configured
to forward Provide and Register transactions to the endpoint documented above in the Supporting
Environment Configuration.
