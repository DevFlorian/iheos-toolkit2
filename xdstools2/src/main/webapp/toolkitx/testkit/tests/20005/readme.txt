Multi-Op: Submit Association and AvailabilityStatus (Beta)

Tests the ability of Document Registry to accept a both a Submit Association and Update Availability Status operations for a new Association object when received in a single submission.
 
Beta Test (CP-ITI-1029) - Rules for this test are implied from Section XDS MU-TI 3.57.4.1.3.1 - Common Rules for Metadata Update (Items #11 and #12).
