Folder - Simple Update Using Association Propagation
 
Tests the ability of the Document Registry to accept a Folder that includes two Document Entry objects. This will be followed by the submission of a new version of the Folder using the Folder Metadata Update operation using association propagation.  This exercises the basic operation of the Update Folder Metadata operation as defined in section 3.57.4.1.3.3.3 of the Metadata Update Supplement.
