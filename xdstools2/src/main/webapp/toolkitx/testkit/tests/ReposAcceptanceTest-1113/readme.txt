Folder and Document Entry Tests

SubmitTwoDocs submits two-documents.

SubmitFolder submits a Folder.

SubmitFolderwDoc submits a Folder and a Document Entry. The Document Entry is associated with the Folder.

AssociateNewDEwithExistingFolder is a submission of a Document Entry with an association to an existing Folder. (A reference to the Folder is taken from the SubmitFolder section.)

AssociateExistingDEwithExistingFolder is a submission that associates an existing Document Entry with an existing Folder. (A reference to the Document Entry is taken from the SubmitTwoDocs section and a reference to the Folder is taken from the SubmitFolder section.)
