ITI-62: Remove FD Object

Test a Document Registry's ability to remove a SubmissionSet containing a single empty Folder.

The exceptions, ReferencesExistException and XDSUnreferencedObjectException, are included in this test evaluation. 

Reference: ITI TF-2c: Section 3.62.4.1.3 - Expected Actions

Note: This is the same as Test 30001, except for the submitted metadata object.
