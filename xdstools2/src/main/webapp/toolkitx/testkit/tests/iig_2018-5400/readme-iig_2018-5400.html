Initiating Imaging Gateway (2018): Read This First

The System Under Test (SUT) is an Initiating Imaging Gateway.

Testing Overview

The tests for the Initiating Imaging Gateway fall into several broad categories and are numbered accordingly. The numbering scheme and categories are listed in the following table.

Test Number Base Test Category
iig_2018-5401x
  • Testing with a single Respnding Imaging Gateway
  • These are cases where success responses are intended
iig_2018-5402x Success cases involving different DICOM transfer syntaxes
iig_2018-5403x Success cases involving multiple Responding Imaging Gateways
iig_2018-5404x Failure cases
iig_2018-5405x Partial success cases with single and multiple Responding Imaging Gateways

Configuration

You need to configure your Initiating Imaging Gateway to communicate with simulators listed in the conformance testing tool. If you are reading this document as a standalone document, that configuration is not available to you. If you are reading this document in the context of the XDS Toolkit, it is better to close this test and read the introductory material in the toolkit for testing the Imaging Document Source. That introductory text includes configuration parameters and diagrams.
