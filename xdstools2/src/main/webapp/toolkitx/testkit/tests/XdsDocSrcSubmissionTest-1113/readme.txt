Folder and Document Entry Tests

SubmitTwoDocs looks for a two-document submission.

SubmitFolder looks for a Folder submission.

SubmitFolderwDoc looks for a Folder and a Document Entry submission with the Document Entry submitted as part of the Folder.

AssociateNewDEwithExistingFolder looks for a submission that adds a Document Entry to the Registry and adds it to an existing Folder in the same operation.

AssociateExistingDEwithExistingFolder looks for a submission that associates an existing Document Entry with an existing Folder.
